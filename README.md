# Kiln

## Local Skill Install

Use `scripts/install-safe-solana-builder.sh` to sync the bundled Solana skill from `context/safe-solana-builder-main` into your local Codex and/or Claude skills folders.

```bash
scripts/install-safe-solana-builder.sh
scripts/install-safe-solana-builder.sh --claude
scripts/install-safe-solana-builder.sh --both
```
